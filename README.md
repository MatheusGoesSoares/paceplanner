# PacePlanner

PacePlanner é uma aplicação web fullstack desenvolvida para registrar e gerenciar treinos de corrida de forma prática e moderna. O projeto foi criado como parte da atividade Desafio IV - Fullstack, aplicando conceitos de frontend, backend, banco de dados, PWA e deploy em nuvem.

## Aplicação Online

Frontend: https://seu-link-netlify.netlify.app

Backend API: https://paceplanner-backend.onrender.com

## Repositório

https://github.com/MatheusGoesSoares/paceplanner

## Funcionalidades

- Cadastro de treinos de corrida
- Visualização do histórico de treinos
- Edição de registros
- Exclusão de treinos
- Captura de localização pelo dispositivo
- Interface responsiva para computador e celular
- Aplicação instalável como PWA
